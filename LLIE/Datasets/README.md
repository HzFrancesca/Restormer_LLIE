# Datasets

Please place the LOL-v2 dataset in the `datasets` directory at the root of the project. The structure should look like this:

```
Restormer/
├── datasets/
│   ├── LOL-v2/
│   │   ├── Real_captured/
│   │   │   ├── Train/
│   │   │   │   ├── Low/
│   │   │   │   └── Normal/
│   │   │   └── Test/
│   │   │       ├── Low/
│   │   │       └── Normal/
...
```
